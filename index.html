Oral Cancer Detection
CNN & Vision Transformer

Binary image classification of oral cancer using a custom CNN baseline and a Vision Transformer (ViT) built from scratch in TensorFlow/Keras.

📂 View on GitHub 📓 Open in Kaggle
Python 3.10 TensorFlow 2.x Keras Scikit-learn ✓ Complete
📖 Project Overview
Early detection of oral cancer using deep learning on lip and tongue images
🎯

Objective

Automate binary classification of oral images into cancer and non-cancer categories to assist early clinical screening.

🧠

Two Architectures

Baseline custom CNN with regularization, and a Vision Transformer (ViT) built entirely from scratch using patch embeddings and multi-head attention.

⚖️

Imbalanced Data Handling

Sklearn class weight balancing and data augmentation (flip, rotation, zoom) applied to improve generalization on a small 131-image dataset.

📊

Evaluation

Confusion matrix, accuracy curves, and training/validation loss plots to monitor performance and detect overfitting.

🏗️ Model Architectures
Custom CNN baseline and ViT from scratch

🔷 Baseline CNN

Input (240×240×3)
↓ Conv2D(32, 3×3, stride=2) + ReLU
↓ MaxPooling2D(2×2)
↓ Conv2D(32, 3×3, stride=2) + ReLU
↓ MaxPooling2D(2×2)
↓ Flatten
↓ Dense(64) + L1L2 + Dropout(0.5)
↓ Dense(1) + Sigmoid

🔶 Vision Transformer (ViT)

Input (224×224×3)
↓ Data Augmentation
↓ Patch Extraction (16×16 → 196 patches)
↓ Linear Projection + Positional Embeddings
↓ Transformer Encoder × 8 layers
   (MHA → LayerNorm → MLP → LayerNorm)
↓ MLP Head (2048 → 1024)
↓ Dense(1) + Sigmoid
